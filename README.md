Shops App using React Native and AWS AppSync
============================================

How to use:
-----------

Clone this project
------------------
```
git clone https://github.com/redjohnzarra/shops-appsync.git
```
Go to the cloned project
------------------------
```
cd shops-appsync
```
Install node modules
--------------------
```
npm install or yarn install
```
Run the project
---------------
```
npm start or yarn start
```
